Memelysis is a website with meme analytics and a place to find your favourite memes. It consists of memes from four sources: Reddit, Twitter, Imgur and Memedroid.

It was created as a project for Big Data course (summer semester 2019/2020) at Warsaw University of Technology by Michał Dyczko, Jakub Kała, Maciej Pawłowski and Krzysztof Spaliński. Source code can be found here.
